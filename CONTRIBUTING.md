# Contributing

1. Create an issue for the task.
2. Create a branch from `develop`.
3. Use clear commit messages.
4. Run linting, tests, and builds locally.
5. Open a pull request.
6. Ask the other developer for review.
7. Merge only after checks pass.

Commit examples:

```text
feat(api): add investigation health endpoint
feat(web): display backend status
fix(db): correct custody event index
docs(setup): clarify Docker startup
test(api): add database health test
```
